Qubit has extensive support documentation to help people use the product to its full potential. As the platform caters for a diverse range of professions from javascript engineer to marketing associate, Qubit has to provide guidance from simple WYSIWYG Ui editing tasks all the way to complex javascript implementation of A/B tests. The challenge is finding an accessible way to display all this information.
The solution to helping users find the information they need as easly as possible was self segmentation. This allows the most relevant information to be shown, based on the users profession.
Information is organised based on the Qubit features users may want to action on. As each feature includes multiple setting options and activities, the information is orderd by ’tasks’. The tasks are then split up into actionable steps.
Code snippets and Ui examples are utilized to make the information as useful and legible as possible. Recently viewed articles are surfaced to help users navigate back to tasks they may want to continue.
